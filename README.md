## 📌 Executive Summary

**Agentic Studio Hub** is an all-in-one, browser-native AI production suite designed to democratize high-end 3D motion graphics, visual effects (VFX), video editing, and agentic code execution for independent content creators, YouTubers, and digital media teams.

Professional creative suites (Cinema 4D, Maya, After Effects) require $10,000+ in software licenses, high-end GPUs, and **15 to 20 hours of manual keyframing** per video. **Agentic Studio Hub** eliminates this barrier by replacing manual workflows with autonomous AI director agents powered by **GPT-5.6** and **OpenAI Codex**, cutting production time by **90%** at zero software subscription cost.

## ✨ Core Suite Modules

```
 💼 AGENTIC STUDIO HUB (Unified Dashboard)
 ├── 🎨 1. Agentic 3D Studio (Prompt-to-3D, Mograph Shaders, Tank Charts)
 ├── 🎬 2. Agentic Video Editing (Reels Auto-Cutter, Transcript Scoring, 9:16 Shorts)
 ├── ⚡ 3. Agentic Coding Studio (Codex Scripting, WebGL Memory Pipeline)
 └── 🚀 4. Production Product Hub (System Management & Crawl Automation)
```

### 1. 🎨 Agentic 3D & Motion Graphics Studio
* **Prompt-to-Scene Generation:** Describe 3D scenes in plain text (e.g. *"Glowing futuristic neon particle ring rotating around a metallic sphere"*).
* **`mograph.mjs` Shader Engine:** Codex generates procedural WebGL motion graphics scripts in real time.
* **Procedural Data Visualizations:** Generates 3D tank comparison charts, bar charts, and solar system orbits automatically.

### 2. 🎬 Agentic Video Editing & Shorts Engine
* **Long-to-Short Auto-Cutter:** Analyzes long-form MP4/MOV videos and extracts high-engagement vertical 9:16 clips for TikTok, Shorts, and Reels.
* **AI Moment Scoring:** GPT-5.6 ranks video segments based on Hook Strength, Retention Value, and Audience Trend potential.
* **Interactive Timeline Seeking:** Click any generated clip card to jump to exact video timestamps instantly.

### 3. ⚡ Agentic Coding & Incident Memory Studio
* **Live Codex Editor:** Write or prompt JavaScript code directly in the integrated editor and execute it on a live WebGL canvas.
* **Graph RAG Context Injection:** Connects to Cognee Knowledge Graph memory to maintain context-aware entity retention (`[THREE.TorusKnotGeometry, MeshPhysicalMaterial]`) with zero hallucination risk.

## 🛠️ Architecture & Tech Stack

* **Frontend UI:** HTML5, Modern Glassmorphism CSS3 System, JavaScript (ES6+), FontAwesome Icons.
* **3D Engine:** Three.js (r128), WebGL, Custom Shader Utilities (`scenes/mograph.mjs`).
* **AI Models:** OpenAI Codex, GPT-5.6 API, Cognee Graph RAG Memory Engine.
* **Video Engine:** Native HTML5 Video API, Automated FFmpeg timeline slicer bindings.
* **Server Runtime:** Node.js ES Modules (`server.mjs`).

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Ariya-rithvik/Agentic_studio_hub.git
cd Agentic_studio_hub
```

### 2. Run the Local Node.js Server
No heavy installations required! Run using Node.js:
```bash
node server.mjs
```

### 3. Open in Browser
Navigate to:
👉 **[http://localhost:3030](http://localhost:3030)**

*(Alternatively, you can open `index.html` directly in any web browser).*
