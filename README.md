# 🚀 Agentic Studio Hub — Unified 3D, Video & AI Coding Suite

A unified production product homepage and creative studio dashboard that combines:
1. **Agentic 3D Studio & Motion Graphics** (Scene generation, `mograph.mjs`, Three.js WebGL viewports, tank charts).
2. **Agentic Video Editing & Reels AI** (Transcribing, scoring hook/flow/value, vertical 9:16 shorts auto-cutter).
3. **Agentic Coding & Memory Pipeline** (OpenAI Codex live code synthesis, Cognee Knowledge Graph memory, WebGL incident simulations).

---

## ⚡ How to Run

### Option 1: Direct File Open
Open `file:///d:/agentic_studio_hub/index.html` in your web browser.

### Option 2: Node.js Local Server
Run the following command:
```bash
node d:\agentic_studio_hub\server.mjs
```
Then visit: `http://localhost:3000`
